# C Web Parser ![CI status](https://img.shields.io/badge/build-passing-brightgreen.svg)

Projeto para a matéria de supercomputação 2018.2/Insper.

## Installation
cmake . && make

### Requirements
* Linux
* Python 3.3 and up
* iPython Notebook
* mprof
* gprof
* LibCurl
* cmake
* json for modern c++
* gumbo-parse
